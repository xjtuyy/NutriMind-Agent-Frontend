# NutriMind 前端

Vue 3 + Vite 前端，当前按普通用户和管理员两种身份提供不同工作区。

## 启动

```powershell
npm install
npm run dev
```

开发服务器运行在 `http://localhost:3000`，默认将 `/api` 代理到 `http://localhost:9999`。如果后端地址不同，复制 `.env.example` 为 `.env` 后修改 `VITE_API_TARGET`。

## 已实现功能

- 用户注册、登录、身份恢复、修改密码和退出登录
- 根据 `is_superuser` / 管理员角色进入对应工作区
- 上传知识库文档（PDF、Markdown、TXT、TEXT）
- 语义检索、按来源删除文档、知识库统计
- 拍照上传、上传历史、图片查看和删除
- 管理员 Dashboard 聚合数据展示
- 开发环境下的用户端和管理员端无后端预览
